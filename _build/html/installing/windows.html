Windows¶

To get started with Ergo on Windows:

  1. Make sure you have the latest release downloaded.

  2. Extract the zip file to a folder.

  3. Copy and rename default.yaml to ircd.yaml.

  4. Open up ircd.yaml using any text editor, and then save it once you’re happy.

  5. Open up a cmd.exe window, then cd to where you have Ergo extracted.

  6. Run ergo mkcerts if you want to generate new self-signed SSL/TLS certificates (note that you can’t enable STS if you use self-signed certs).

To start the server, type ergo run and hit enter, and the server should start!
