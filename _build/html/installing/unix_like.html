macOS / Linux / Raspberry Pi¶

To get started with Ergo on macOS, Linux, or on a Raspberry Pi:

  1. Make sure you have the latest release for your OS/distro downloaded.

  2. Extract the tar.gz file to a folder.

  3. Copy and rename default.yaml to ircd.yaml.

  4. Open up ircd.yaml using any text editor, and then save it once you’re happy.

  5. Open up a Terminal window, then cd to where you have Ergo extracted.

  6. Run ./ergo mkcerts if you want to generate new self-signed SSL/TLS certificates (note that you can’t enable STS if you use self-signed certs).

To start the server, type ./ergo run and hit enter, and the server should be ready to use!
